Smart Calc Documentation

Introduction

Smart Calc is a powerful and easy-to-use calculator that supports basic arithmetic operations, functions, and stack operations.

Data Structures

StringStack

StringStack is a stack data structure that stores strings. It is used to store and manipulate mathematical expressions.

DoubleStack

DoubleStack is a stack data structure that stores doubles. It is used to store and manipulate numerical values.

Functions

create_string_stack()

Creates a new StringStack object.

is_string_stack_empty(stack)

Checks if the StringStack object is empty.

push_string(stack, data)

Pushes a string onto the top of the StringStack object.

pop_string(stack)

Pops and returns the top string from the StringStack object.

peep_string(stack)

Returns the top string from the StringStack object without popping it.

print_string_stack(stack)

Prints the contents of the StringStack object.

free_string_stack(stack)

Frees the memory used by the StringStack object.
